# 프로젝트 청사진

## 개요

이 프로젝트는 사용자에게 즐거움을 제공하는 간단한 웹 애플리케이션 모음입니다. 현재는 로또 번호 생성기 기능을 중심으로 개발하여, 사용자가 재미있게 웹사이트를 이용하고 재방문하도록 유도하는 것을 목표로 합니다.

## 스타일, 디자인 및 기능

- **최신 디자인:** 깨끗하고 현대적인 디자인을 적용하여 사용자 경험을 향상시킵니다.
- **반응형 레이아웃:** 모바일과 데스크톱 등 다양한 화면 크기에서 최적의 화면을 제공합니다.
- **상호작용성:** 사용자가 직접 번호를 생성하고 결과를 확인할 수 있는 동적인 기능을 제공합니다.
- **콘텐츠:** 로또 번호 생성기의 사용법과 의미를 설명하는 텍스트 콘텐츠를 포함합니다.

## 현재 요청 계획

1.  **`index.html` 구조 변경:** 로또 번호 생성기 UI와 설명 콘텐츠를 추가합니다.
2.  **`style.css` 디자인 개선:** 현대적인 스타일, 색상, 레이아웃을 적용하여 시각적 매력을 높입니다.
3.  **`main.js` 기능 구현:** 로또 번호 생성 및 표시 로직을 작성합니다.
4.  **`lotto.html` 삭제:** 기존 `lotto.html`의 기능을 `index.html`에 통합하고 불필요한 파일을 정리합니다.
